# SlimefunOreChunks 矿石块

SlimefunOreChunks 是一个 Slimefun4 附属插件。
你需要安装 Slimefun4 才能让此插件运行。

该附属向GEO矿机添加了新的产物：矿石块。一共有9种矿石块，分别对应着9种Slimefun金属，你可以在矿石粉碎机中粉碎矿石块来获得对应的矿粉。

要注意的是，这些矿石并不会自然生成，你只能通过GEO矿机获得它们。
